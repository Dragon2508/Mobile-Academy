Создание проекта в Visual Studio

Пошаговые инструкции для Windows:

1. Запустите Visual Studio 2019. Для создания нового проекта, необходимо выбрать Создание проекта:

2. В создание проекта выполните поиск по слову "Xamarin" и выберите тип проекта Мобильное приложение (Xamarin.Forms) :

3. В настройке нового проекта введите имя проекта и его расположение. Нажмите кнопку Создать в правом нижнем углу :

4. Выберите тип проекта Пустой, убедитесь, что выбраны параметры Android и iOS и нажмите кнопку Создать:
